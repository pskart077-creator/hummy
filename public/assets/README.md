# Assets Hummy Original

Use esta pasta para arquivos publicos do site.

- `brand`: logos, simbolos e arquivos da identidade visual.
- `products`: fotos, renders e mockups dos produtos.
- `banners`: imagens de hero, campanhas e faixas promocionais.
- `icons`: icones exportados como SVG ou PNG.
- `social`: imagens para Open Graph, redes sociais e compartilhamento.
- `payments`: selos, bandeiras e imagens de pagamento.

Arquivos dentro de `public/assets` sao acessados pela URL `/assets/...`.
